# 贴图小王子图床服务器端代码
### 使用Node.js搭建，完成了以下任务：
- 接收从贴图小王子App上传的图片，生成缩略图，并将原图和缩略图分别存到不同的文件夹中。
- 支持查询接收到的原图及其缩略图的信息。
- 支持浏览器直接上传图片到服务器。
